# 📚 Notions requises — Double Numérique Intelligent

---

## 🧮 Mathématiques

### Algèbre linéaire

- Vecteurs et espaces vectoriels
- Produit scalaire (dot product)
- Similarité cosinus → **utilisée pour la recherche vectorielle (RAG)**
- Matrices et transformations linéaires

### Probabilités & Statistiques

- Distributions de probabilité
- Espérance, variance
- Bayes naïf (compréhension des prédictions LLM)
- Entropie → **utilisée dans les modèles de langage**

### Calcul différentiel (notions de base)

- Dérivées et gradients → **comprendre comment les LLMs apprennent**
- Descente de gradient (concept, pas implémentation)

---

## 🧠 Intelligence Artificielle

### LLM & Prompt Engineering

- Tokenisation
- Température, top-p, max_tokens
- Prompt structuré (system / user / assistant)
- Few-shot prompting
- Chain of Thought (CoT)

### RAG (Retrieval-Augmented Generation)

- Embeddings — transformer du texte en vecteur
- Chunking — découper les documents
- Recherche par similarité cosinus
- Reranking — trier les résultats pertinents

### Agents & Planification

- ReAct (Reasoning + Acting)
- Plan-and-Execute pattern
- Tool calling / Function calling
- Mémoire d'agent (buffer, summary, vectorielle)

---

## 💻 Programmation

### Python

- Typage statique (type hints, Pydantic)
- Async / Await (asyncio)
- Décorateurs
- Gestion des exceptions
- Variables d'environnement (python-dotenv)

### API & Backend

- REST API (méthodes HTTP, status codes, headers)
- FastAPI (routing, dépendances, middleware)
- Authentification JWT (access token, refresh token)
- WebSocket (communication temps réel)

### Bases de données

- SQL (SELECT, INSERT, JOIN, index)
- ORM (SQLAlchemy — modèles, sessions, migrations)
- Alembic (migrations PostgreSQL)
- Redis (clé-valeur, TTL, pub/sub)
- Base vectorielle (Chroma — collection, embedding, query)

### Architecture logicielle

- Séparation des responsabilités (SRP)
- Injection de dépendances
- Pattern Repository (accès données)
- Middleware
- Variables d'environnement et configuration

### Frontend (Next.js)

- React hooks (useState, useEffect, useRef)
- Fetch API / Axios
- WebSocket côté client
- Tailwind CSS (utility-first)
- TypeScript (types, interfaces, generics de base)

### DevOps (Phase 1)

- Docker (image, container, volume)
- Docker Compose (services, dépendances, réseaux)
- Variables d'environnement en production

---

## 🔐 Sécurité (Phase 2+)

- Hashing de mots de passe (bcrypt)
- JWT — structure, signature, expiration
- CORS
- Isolation de données multi-utilisateurs
- Sandbox d'exécution de code
